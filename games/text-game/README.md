# Nirvana Text Adventure Game

Welcome to the Nirvana Text Adventure Game! This is an interactive text-based game designed to immerse you in a unique gaming experience.

## Introduction
Nirvana is a text adventure game where players navigate through a rich storyline by making choices that affect the outcome of the game. 

## Gameplay
- **Explore** different environments and scenarios.
- **Make decisions** that shape your story.
- **Solve puzzles** and overcome challenges to progress.

## Installation
To play the Nirvana Text Adventure Game, clone the repository and navigate to the game directory:
```bash
git clone https://github.com/spaceegrace/portfolio.git
cd portfolio/games/text-game
```
